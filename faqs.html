General

Can I use this with an existing project?

Technically not, this is meant to be the foundation for a new project or rebuild. Obviously it is up to you whether or not you decide to port the files over to an existing project, but I won't be responsible for any flaws created :)

Is this project for beginners?

While this project is aimed toward anyone, basic knowledge of the framework as a whole is required. It is recommended you read the Laravel documentation in full before installing this project.
